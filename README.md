# 📸 LokaLensa Kids

**LokaLensa Kids** adalah landing page website edukasi kreatif untuk anak-anak yang ingin belajar dunia digital secara menyenangkan. Website ini menghadirkan pengalaman belajar seperti permainan, dengan desain modern, warna ceria, dan interaksi yang ramah anak.

Platform ini memperkenalkan anak-anak pada berbagai keterampilan digital seperti:

- Fotografi
- Editing Video
- Coding Game

---

# ✨ Fitur Utama

## 🎨 Desain Ramah Anak
Tampilan website menggunakan warna ceria dan tipografi yang mudah dibaca sehingga cocok untuk anak-anak.

## 📱 Responsive Design
Website dapat diakses dengan baik melalui:
- Desktop
- Tablet
- Smartphone

## 🎭 Mentor Interaktif
Setiap mentor memiliki profil yang dapat dibuka melalui **modal popup** untuk melihat detail mentor.

## ⚡ Animasi Modern
Website menggunakan animasi seperti:
- Scroll animation
- Floating animation
- Button press effect

## 🧩 Struktur Website Modular
Kode dipisahkan agar mudah dikembangkan:

- **HTML** → Struktur halaman
- **CSS** → Styling dan animasi
- **JavaScript** → Interaksi pengguna

---

# 🛠 Teknologi yang Digunakan

Website ini dibangun menggunakan teknologi berikut:

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- TailwindCSS
- Font Awesome
- Google Fonts
